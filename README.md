# investment-committee
An AI tool that simulates an investment committee meeting using Delphi method, which allows you to see the risks and rewards in real-time.  
Implementation via with Google ADK.

# Setup
From the project root:
For working with python venv:
```
python -m venv .venv
source .venv/bin/activate
pip install -r backend/requirements.txt
```

# Start UI
```
adk web
```
Write a stock ticker in the prompt box (e.g. AAPL) and press enter.
